# How to run the Project
1. Download and Unzip the file on your local system.
2. Put the unzipped folder inside root directory
-- For XAMPP go to xampp/htdocs and put the unzipped folder there --

# Localhost configuration
1. open or download XAMPP 
2. start APACHE and MySQL 

# Database Configuration
1. Open PHPMYADMIN
2. Create Database rentcar_project
3. Import database rentcar_project.sql (available SQL File inside zip package/database folder)

# View the website:
1. navigate to the following link on the localhost
http://localhost/Rent_a_car_project/
